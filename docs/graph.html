        AI-->>U: Typewriter effect

⚙️ Environment Configuration (.env)

All knobs you can turn
VariableValueDescription
AI_PROVIDER ollama AI backend: ollama (local), openrouter (cloud), or builtin
OLLAMA_BASE_URL http://127.0.0.1:11434 Ollama server address (used when AI_PROVIDER=ollama)
OLLAMA_MODEL llama3.2:latest Model name for Ollama queries
OPENROUTER_API_KEY sk-... API key for OpenRouter (optional, uncomment to use cloud models)
OPENROUTER_MODEL openai/gpt-4o-mini Model identifier on OpenRouter
OBSIDIAN_VAULT_PATH (default: ./OpenFriday) Custom path to your Obsidian vault directory
